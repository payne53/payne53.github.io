BIGO

一面

  • topk 算法复杂度多少？如果有 1 亿个数据要处理，怎么进行加速？
    • 分块处理再合并
    • 假设分成 a 块，每块数据规模为 b，整体时间可以看成 b + a * k，当 b==a*k的时候比较高效，可以作为一个大概估计
  • BN 和 Dropout 的操作方式
  • epoch 的概念是什么，训练神经网络为什么要多个 epoch？训练多个 epoch 对 神经网络和 SVM 效果有什么不同？
    • 神经网络一个 epoch 未必能达到收敛；
    • 每个 epoch 都会重新 shuffle，产生不同的 batch，相当于数据增强
    • SVM 求解的是一个最优超平面，直观上感觉多个 epoch 没啥意义
  • 一批视频流，每个视频 id 可能重复出现，如何实现一个采样，使得对于每个视频 id 采样的概率比较接近？
    • 基于蓄水池抽样的改进，问题本身没有定义好，只能说一个大概思路
  • 机器学习算法了解多少？Adaboost？GBDT？XGBOOST？

小马智行

一面

  • 给定一个长度为S的链表：

    N(0) -> N(1) -> N(2) -> … -> N(s-2) -> N(s-1)
    将链表原地调整为以下顺序：

    N(0) -> N(s-1) -> N(1) -> N(s-2) -> …. N(s/2-1) -> N(s/2) s 为偶数
    N(0) -> N(s-1) -> N(1) -> N(s-2) -> … N(s/2) s 为奇数
    要求实现函数：

    void ResortNodeList(Node* root) ;

  • Description
    输入n, 表示节点数, 以及若干条边(a, b), 表示a是b的父节点. 判断这些节点是否构成一棵二叉树. 如输入 表示有3个节点, 其中节点0是节点1和节点2的父节点, 3个节点构成了一二叉树.

    要求实现函数：

    bool IsBinaryTree(int n, const std::vector<std::pair<int, int>>& edges) (C++)
    def is_binary_tree(n, edges) (Python)
