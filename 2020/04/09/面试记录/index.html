依图

一面

  • 给定一个包含了一些0（代表水）和1 (代表土地)的非空二维数组 grid。一个 岛屿 是由一些相邻的 1构成的组合，这里的相邻要求两个1必须在水平或者竖直方向上相邻。假设 grid的四个边缘都被0包围。找到给定的二维数组中最大的岛屿面积。(如果没有岛屿，则返回面积为0 )

    例：

    [[0,1,0,1],

    [1,1,1,0],

    [0,0,1,0]] => 5

  • 求正整数数组的最大可能和，要求不能选择数值上相邻的数，比如选择了数字n，就不能选择数字n-1和n+1

    例：[2,2,3,3,4]=>8

  • 6个人，他们之间两两相互认识，或者相互不认识。证明，一定存在3个人，他们之间相互认识，或者相互不认识。

  • 有N个1080P的视频，每次从中随机选择M个视频，从每个视频中随机选择连续的K帧，并在这K帧中随机crop出256x256的patch。请设计一个算法，使得每次得到的连续帧的patch差异尽可能的大。

  • 提问：主要做什么？团队什么情况？

    • 研究三组，主要做语言、文本、安防、交通、人体识别、医疗
    • leader 是人脸识别第一，团队大几十人

二面

  • 问简历，头条的实习

  • 算法题：

    • input: list probs (sum=1, >=0), float cutoff (0,1)
    • output: float sum
    • sum:
      选择最少的数, 使得这些数的和>=cutoff
      返回这些数的和
      如果多解, 让和最大
    • [0.4 0.2 0.3 0.1]
      cutoff
      0.3 -> 0.4
      0.5 -> 0.7
    • 退化了如何优化复杂度？
